# Endee AI Project 🚀

## 📌 Project Title
EV Demand Prediction

## 🧠 Description
This project predicts future demand for electric vehicles (EVs) using a simple AI-based approach. 
It uses historical data and a growth-based model to estimate future demand.

## ⚙️ Features
- EV demand prediction
- User input for year
- Data processing using pandas
- Simple AI logic

## 🛠️ Tech Stack
- Python
- Pandas

## ▶️ How to Run
1. Install pandas:
   pip install pandas

2. Run the program:
   python app.py

## 📈 Output
The program takes a year as input and predicts EV demand for that year.

## 🔗 Note
This project is built using the Endee repository as part of the selection task.
It can be extended further using AI and vector database features.
